Git und GitHub

Ein einfacher Vergleich: Was ist das Programm, was ist die Plattform?

Git

  • Git ist ein Programm für Versionsverwaltung.
  • Es speichert Änderungen an deinem Code.
  • Du kannst ältere Versionen wiederfinden.
  • Git funktioniert auch ohne Internet auf deinem Computer.

GitHub

  • GitHub ist eine Website und Plattform im Internet.
  • Dort kannst du Git-Projekte online speichern.
  • Andere Menschen können deinen Code sehen oder mitarbeiten.
  • GitHub nutzt Git, ist aber nicht dasselbe wie Git.

Der Unterschied kurz gesagt

Git ist das Werkzeug auf deinem Computer. GitHub ist der Ort im Internet, wo du dein Git-Projekt hochladen und teilen kannst.

Git = Werkzeug · GitHub = Online-Plattform
